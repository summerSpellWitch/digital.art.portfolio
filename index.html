Glitch Art

The above images were made by opening them in a text editor like Notepad or Notepad++. The effect was created by deleting or moving portions of the text around which results in a partially corrupt or glitchy looking image. This method has the side effect of possibly fully corrupting the image, forcing one to restart from the beginning.

Allbum Cover

Here is an album cover I made. The art isn't mine, I merely added various visual effects to get the resulting image. Using one of the glitch images from the previous section, I imported it into Photoshop and used different layer types to create the final image

Vector Drawing Sticker

Here is a vector drawing created entirely within Adobe Illustrator using only the pen tool. A reference image was used then imported into Illustrator where the "Image Trace" function was used. Using the pen tool to basically copy the image trace, correctly layering the shapes created with the pen tool, and color picking from the reference image the result is the image above.

Wire sculpture here

This is a sculpture made out of 12-gauge wire using a sword from the game Minecraft as the reference. It was made using pliers and wirecutters to bend and cut the wire into the appropriate shape. Two silhouettes of the sword's overall shape were first made then sandwhiched ontop of each other. More wire was then used to hold them in place and form the criss-crossing sections.

Wood "Puzzle" Set

Pictured above is a wooden "puzzle" cut from a piece of pine wood with a mineral oil finish. The idea behind it was to create something with a sense of direction / flow, like a containted ball of wind. Its creation started with sketches which were then iterated into the final design as seen above. The final design was then pictured, uploaded and traced over in Adobe Illustrator for clarity, then printed out to be used as a template for the wood cutting. The pieces were then sanded smooth and given a rounded edge.
