Glitch Art

The above images were made by opening them in a text editor like Notepad or Notepad++. The effect was created by deleting or moving portions of the text around which results in a partially corrupt or glitchy looking image. This method has the side effect of possibly fully corrupting the image, forcing one to restart from the beginning.

Allbum Cover

Here is an album cover I made. The art isn't mine, I merely added various visual effects to get the resulting image. Using one of the glitch images from the previous section, I imported it into Photoshop and used different layer types to create the final image

Vector Drawing Sticker

Here is a vector drawing created entirely within Adobe Illustrate using only the pen tool. A reference image was used then imported into Illustrate where the "Image Trace" function was used. Using the pen tool to basically copy the image trace, correctly layering the shapes created with the pen tool, and color picking from the reference image the result is the image above.
